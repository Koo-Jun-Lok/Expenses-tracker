A hybrid mobile application developed for personal finance management. The system allows users to record, manage, and analyze daily financial activities through a mobile-friendly interface.

## Project Overview

The Expense Tracker application was developed under the theme of Personal Finance Management.

The system is designed to help users:
- Record income and expense transactions
- Manage personal financial records
- Monitor spending behaviour
- View reports and statistics
- Improve financial awareness

The application adopts a Hybrid Mobile Application approach using WebView technology integrated with PHP and MySQL.

---

## System Objectives

- Develop a mobile-based expense tracking system using a hybrid application architecture
- Implement complete CRUD functionality for transaction management
- Integrate external APIs such as currency exchange and GPS services
- Provide a responsive and user-friendly interface
- Generate visual financial reports for spending analysis

---

## Main Features

### User Management
- User registration and login
- Password reset functionality
- Session-based authentication

### Transaction Management
- Add new income and expense records
- Edit and delete transactions
- View transaction history

### Financial Analysis
- Summary dashboard
- Statistical charts and reports
- Category-based expense analysis

### Additional Functions
- Multi-currency support
- GPS location tagging
- Receipt image upload
- Search and filter functionality
- Calendar-based transaction view

---

## Technology Stack

| Component | Technology |
|----------|-------------|
| Frontend | HTML, CSS, JavaScript, Bootstrap |
| Backend | PHP |
| Database | MySQL |
| Mobile Platform | Android WebView |
| APIs | Currency Exchange API, OpenStreetMap, GPS Services |

---

## System Architecture

The application follows a client-server architecture:
- The frontend interface is developed using HTML, CSS, JavaScript, and Bootstrap
- PHP is used for backend processing and business logic
- MySQL stores user and transaction data
- Android WebView is used to deploy the web application as a mobile application
