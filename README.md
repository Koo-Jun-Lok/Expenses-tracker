## 🧾 Overview

The **Expense Tracker** is developed under the theme of **Personal Finance Management**.

This application helps users:
- Track daily expenses & income  
- Manage financial records  
- Analyze spending habits  

💡 Built using a **Hybrid Mobile App approach (WebView + PHP + MySQL)** to combine web flexibility with mobile accessibility.

---

## ❗ Problem

Many people still rely on:
- Paper receipts  
- Spreadsheets  
- Memory  

These methods are:
- ❌ Inefficient  
- ❌ Error-prone  
- ❌ Lack insights  

👉 This system solves those issues with a centralized mobile solution.

## 🚀 Features

### ✅ Core Features
- 🔐 Authentication (Login / Register / Reset Password)
- 💰 Expense & Income Management
- 🏷️ Categories & tagging
- 📊 Reports with charts
- 🌍 Multi-currency support
- 📷 Receipt upload
- 📍 Location tracking
- 🔎 Search & calendar view
- 📱 Responsive mobile UI

- OCR scanning  
- Investment tracking  
- Bill reminders
