## 🧾 Overview

The **Expense Tracker** is developed under the theme of **Personal Finance Management**.

This application helps users:
- 📈 Track daily expenses & income  
- 📂 Manage financial records  
- 📊 Analyze spending habits  

💡 Built using a **Hybrid Mobile App approach (WebView + PHP + MySQL)** to combine web flexibility with mobile accessibility.

---

## ❗ Problem Statement

Many people still rely on paper receipts, static spreadsheets, or simply their memory to manage finances. These traditional methods are:
- ❌ **Inefficient** & time-consuming  
- ❌ **Error-prone** - ❌ **Lacking in insights** (hard to visualize spending trends)  

👉 **Solution:** This system centralizes financial tracking into a convenient, automated mobile solution.

---

## 🎯 Objectives

- 📱 Develop a mobile expense tracker utilizing a WebView framework.
- 🔄 Implement full CRUD (Create, Read, Update, Delete) operations.
- 🌍 Integrate external APIs for real-time currency conversion and GPS location.
- 🎨 Design an intuitive, user-friendly, mobile-first interface.
- 📊 Provide financial data visualization for better personal economy insights.

---

## 🚀 Features

### ✅ Core Features
- **🔐 Authentication:** Secure Login / Register / Reset Password system.
- **💰 Transaction Management:** Add, edit, view, and delete income/expenses.
- **🏷️ Smart Categorization:** Visual tags and categories for easy sorting.
- **📊 Reports:** Visual charts and statistical breakdowns.
- **🌍 Multi-Currency:** Real-time exchange rate support.
- **📷 Digital Receipts:** Upload receipt photos for record-keeping.
- **📍 Location Tracking:** Auto-tag transaction locations via GPS.
- **🔎 Search & Filter:** Easy lookup and calendar view integration.
- **📱 Responsive UI:** Optimized for mobile touch interactions.

### ❌ Out of Scope
- Bank account integration  
- Automatic OCR receipt scanning  
- Investment tracking  
- Automated bill reminders  
- Multi-user/household accounts
